# vite 2

## 概念

1. 一个开发服务器
2. 一个构建命令

## 优点

1. 启动项目快呀(不编译)
2. 易上手

## 使用

- 安装

  - npm 安装
    `npm init @vitejs/app`

  - yarn 安装
    `yarn create @vitejs/app`

- vite.config.js 配置

类似之前的vue.config.js，有不同
[vite.config.js](https://vitejs.dev/config/)

- env 环境变量

同样类似之前的vue项目，但是使用方法不同
[env](https://vitejs.dev/guide/env-and-mode.html)
